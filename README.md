# Schlosser Digest

Premium lifestyle website built with Astro and Tailwind CSS.

## 🚀 Deployment

### Vercel (Recommended)

1. Push code to GitHub
2. Go to https://vercel.com/new
3. Import your repository `hsr88/schlosser`
4. Framework Preset: Select **Astro**
5. Build Command: `npm run build`
6. Output Directory: `dist`
7. Add Environment Variable: `NODE_VERSION` = `18`
8. Click **Deploy**

### Manual Deploy

```bash
npm i -g vercel
vercel --prod
```
